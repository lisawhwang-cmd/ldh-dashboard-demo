# LDH Dashboard GitHub Pages Ready

This package is ready for GitHub Pages deployment.

## Files
- `index.html` - dashboard home page. Upload this file to the root of your GitHub repository.
- `README.md` - this instruction file.

## GitHub Pages steps
1. Unzip this ZIP file.
2. Upload `index.html` and `README.md` to the root of your repository.
3. Go to Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main, Folder: / (root).
6. Wait 1-2 minutes, then open: `https://<username>.github.io/<repository-name>/`

Important: `index.html` must be in the repository root. Do not upload only the ZIP file.
